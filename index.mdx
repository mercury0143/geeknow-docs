## 快速开始

```bash
curl -X POST https://api.example.com/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      { "role": "user", "content": "用一句话介绍这个 API" }
    ]
  }'
```

## 路由来源

本文档的文字系列接口来自以下路由文件：

| 文件 | 覆盖内容 |
