# 网页工具接口

网页工具接口位于 `/v1` 下，但不要求请求体中传入模型名称。路由直接进入工具类 relay，适合在 Agent 工作流中补充网页读取和搜索能力。

## 网页读取

```http
POST /v1/web-reader/read
```

<ParamField body="url" type="string" required>
需要读取的网页 URL。
</ParamField>

<ParamField body="format" type="string">
期望输出格式，例如 `markdown`、`md`、`text`、`txt`。默认通常为 `md`。
</ParamField>

<ParamField body="liteMode" type="boolean">
是否启用精简模式。
</ParamField>

<ParamField body="includeImages" type="boolean">
是否在读取结果中包含图片信息。
</ParamField>

<ParamField body="onlyCSSSelectors" type="array<string>">
只保留匹配指定 CSS 选择器的元素。
</ParamField>

<ParamField body="waitForCSSSelectors" type="array<string>">
等待指定 CSS 选择器出现后再解析页面。
</ParamField>

<ParamField body="excludeCSSSelectors" type="array<string>">
排除匹配指定 CSS 选择器的元素。
</ParamField>

<ParamField body="linkSummary" type="boolean">
是否把页面链接汇总到内容末尾。
</ParamField>

<ParamField body="ignoreCache" type="boolean">
是否忽略缓存并重新读取。
</ParamField>

<ParamField body="readTimeout" type="integer">
页面读取超时时间，单位为秒。
</ParamField>

<ParamField body="topic" type="string">
只提取与指定主题相关的内容。
</ParamField>

<RequestExample>

```bash
curl -X POST https://api.example.com/v1/web-reader/read \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/blog/api-gateway",
    "format": "markdown",
    "includeImages": true,
    "readTimeout": 20
  }'
```

</RequestExample>

## 网页搜索

```http
POST /v1/web-search/search
```

<ParamField body="query" type="string" required>
搜索关键词或自然语言问题。
</ParamField>

<ParamField body="area" type="string">
搜索地区，例如 `global`、`cn`。默认通常为 `global`。
</ParamField>

<ParamField body="freshness" type="string">
结果时效性，例如 `Day`、`Week`、`Month`、`Year`。
</ParamField>

<ParamField body="includeImages" type="boolean">
是否同时返回图片搜索结果。
</ParamField>

<ParamField body="page" type="integer">
页码，默认通常为 `1`。
</ParamField>

<ParamField body="count" type="integer">
每页结果数量，常用范围 `1` 到 `50`。
</ParamField>

<ParamField body="format" type="string">
返回格式，例如 `json`、`markdown`、`md`、`text`、`txt`。
</ParamField>

<RequestExample>

```bash
curl -X POST https://api.example.com/v1/web-search/search \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "OpenAI Responses API 多模态 工具调用",
    "area": "global",
    "count": 5,
    "format": "json"
  }'
```

</RequestExample>
