# 特惠渠道系列

特惠渠道系列当前包含 `manxue-2.0.0` 视频模型。该模型通过 `/v1/videos` 创建异步任务，通过 `/v1/videos/{id}` 查询任务状态和结果。

## 模型

| 模型 | 可用 `model` | 请求格式 | 时长 | 清晰度 | 参考图 |
| --- | --- | --- | --- | --- | --- |
| manxue-2.0.0 | `manxue-2.0.0` | JSON | `4` 到 `15` 秒 | `480P`、`720P` | 图片 URL 放入 `content[].image_url.url` |

## manxue-2.0.0 请求特点

- `model` 固定传 `manxue-2.0.0`。
- 使用 `application/json` 提交。
- 时长支持 `4` 到 `15` 秒。
- 清晰度支持 `480P`、`720P`。
- 支持 `16:9`、`9:16`、`1:1`、`3:4`、`4:3`、`21:9`。
- 参考图不接收 Base64 data URI，应传入服务端可访问的公网图片 URL。
- 参考图在 `content` 中使用 `image_url` 项，并通过 `role` 区分 `first_frame`、`last_frame`、`reference_image`。

## 查询任务

任务提交成功后会返回任务 ID。后续使用同一个查询接口：

```http
GET /v1/videos/{id}
```

任务完成后读取 `video_url`；如果接口没有直接返回公开视频地址，可使用 `GET /v1/videos/{id}/content` 下载视频内容。
