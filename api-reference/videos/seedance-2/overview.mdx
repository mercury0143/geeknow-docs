# Seedance-2

Seedance-2 相关能力当前包含两类接口：

- 视频任务接口：`POST /v1/videos`、`GET /v1/videos/{task_id}`、`GET /v1/videos/{task_id}/content`
- 素材接口：`POST /api/asset/createMedia`、`GET /api/asset/get`

## 当前模型

- `seedance-2.0-lite`
- `seedance-2.0-pro`
- `seedance-2.0-fast`

## 兼容模型名

你给的 Apifox 还列出了以下兼容模型名：

- `doubao-seedance-2-0-260128`
- `doubao-seedance-2-0-fast-260128`
- `sd2_manxue_720p`

## 请求特点

- `application/json` 与 `multipart/form-data` 都支持。
- `multipart/form-data` 只支持 URL 字段，不支持真实文件上传。
- `prompt` 必填。
- `duration` 优先于 `seconds`。
- `input_reference` / `image` / `image_url` 默认映射为首帧图。

## 素材 ID 使用方式

Seedance-2 素材接口返回的 `Result.Id` 是裸素材 ID，例如 `asset-20260528143012-a8x7k`。

| 步骤 | 调用 | 素材 ID 写法 |
| --- | --- | --- |
| 创建素材 | `POST /api/asset/createMedia` | 响应返回 `Result.Id` |
| 查询素材 | `GET /api/asset/get?id=asset-20260528143012-a8x7k` | 查询参数使用裸 ID |
| 创建视频 | `POST /v1/videos` | 素材字段使用 `asset://asset-20260528143012-a8x7k` |

`asset://{asset_id}` 可以用于 `first_image`、`last_image`、`reference_images`、`reference_videos`、`reference_audios` 及其兼容字段。建议在 `Result.Status` 为 `Active` 后再把素材用于视频生成。
