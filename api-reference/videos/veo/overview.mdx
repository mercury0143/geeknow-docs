# Veo 视频

Veo 系列使用 `POST /v1/videos` 和 `GET /v1/videos/{task_id}`，请求为 JSON 格式。

- 使用 `POST /v1/videos` 提交视频生成任务。
- JSON 参考图建议使用 `image` 或 `images`；上传本地图片时使用 `input_reference`。
- 模型包括 `veo_3_1`、`veo_3_1-fast`。

## 当前模型

- `veo_3_1`
- `veo_3_1-fast`

## 当前接口行为

| 项目 | 当前行为 |
| --- | --- |
| 请求格式 | JSON |
| 常见输入字段 | `model`、`prompt`、`size`、`duration`、`image`、`images` |
| 上传参考图字段 | `input_reference` |
| 参考生视频约束 | 实际效果取决于当前模型支持 |
| 查询接口 | `GET /v1/videos/{task_id}` |
