# Veo 视频

Veo 系列使用 `POST /v1/videos` 和 `GET /v1/videos/{task_id}`，请求为 JSON 格式。

- 入口路由仍然是 `POST /v1/videos`。
- 参考图会先放进 `input_reference`，再由服务端映射到 Veo 请求结构。
- 模型包括 `veo_3_1`、`veo_3_1-fast`。

## 当前模型

- `veo_3_1`
- `veo_3_1-fast`

## 当前主链路行为

| 项目 | 当前行为 |
| --- | --- |
| 请求格式 | JSON |
| 常见输入字段 | `model`、`prompt`、`size`、`seconds`、`input_reference`、`metadata` |
| 参考图字段 | `input_reference` |
| 参考生视频约束 | 当前参考生视频模式会强制约束到 `16:9` |
| 查询接口 | `GET /v1/videos/{task_id}` |
