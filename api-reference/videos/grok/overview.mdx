# Grok 视频

Grok 视频系列主要围绕 `POST /v1/videos` 和 `GET /v1/videos/{task_id}` 这一组接口展开。

- `grok-video-1.5` / `grok-video-3` 使用 `multipart/form-data`，参考图通过 `input_reference` 传入。
- `grok-imagine-video-1.5-preview` 使用 JSON 请求体，参考图通过 `image` / `images` 传入。
- 常用字段是 `model`、`prompt`、`aspect_ratio`、`seconds`、`size` 或 `resolution`。

## 支持模型

| 模型 | 说明 | 时长规则 |
| --- | --- | --- |
| `grok-video-1.5` | Grok 1.5 | 支持 `10s` / `15s`，分别对应 Pro / Max 档 |
| `grok-video-1.5-pro` | 1.5 Pro 版，支持文生视频、首帧生视频、首尾帧、参考生视频 | 固定 `10s` |
| `grok-video-1.5-max` | 1.5 Max 版，支持文生视频、首帧生视频、首尾帧、参考生视频 | 固定 `15s` |
| `grok-video-3` | Grok 3 | 支持 `10s` / `15s`，分别对应 Pro / Max 档 |
| `grok-video-3-pro` | 3 Pro 版，支持文生视频、首帧生视频、首尾帧、参考生视频 | 固定 `10s` |
| `grok-video-3-max` | 3 Max 版，支持文生视频、首帧生视频、首尾帧、参考生视频 | 固定 `15s` |
| `grok-imagine-video-1.5-preview` | Grok Imagine 1.5 Preview，JSON 提交 | 支持自定义秒数，最小 `1s` |

## 常见能力

| 能力 | 传参方式 | 说明 |
| --- | --- | --- |
| 文生视频 | 仅传文本字段 | 最基础调用方式 |
| 首帧生视频 | `input_reference` 传 1 张图 | 可把首帧图或指定参考图作为第一张输入 |
| 首尾帧 | `input_reference` 传 2 张图 | 按顺序传入首帧、尾帧 |
| 参考生视频 | `input_reference` 传多张图 | Grok 3 最多支持 6 张；Grok 1.5 参考生视频模式仅支持 1 张 |
| Grok Imagine 参考图 | JSON `image` 或 `images` | 单图用 `image`，多图用 `images` |

## 参数约束

| 字段 | 可用值 | 说明 |
| --- | --- | --- |
| `aspect_ratio` | Grok 1.5 / 3 常用 `2:3`、`3:2`、`1:1`；Grok Imagine 额外支持 `16:9`、`9:16` 等 | 建议显式传入目标比例 |
| `size` | `480P`、`540P`、`720P`、`1080P` | 适用于 `grok-video-1.5` / `grok-video-3` |
| `resolution` | `480P`、`720P` | 适用于 `grok-imagine-video-1.5-preview` |
| `seconds` | Grok 1.5 / 3 使用 `10` 或 `15`；Grok Imagine 支持自定义整数秒 | `10s` 对应 Pro 档，`15s` 对应 Max 档 |
