# Sora 视频生成

Sora 当前使用 `POST /v1/videos` 提交任务，并以 JSON 提交为主。

- 路由入口是 `POST /v1/videos`。
- 当前使用 JSON 提交。
- 参考图会先放入 `input_reference`，网关再按上游需要继续处理。
- `router/video-router.go` 里 `/v1/videos` 仍由 `controller.RelayTask` 统一接收。

## 当前模型

- `sora-2`

## 方法与路径

```http
POST /v1/videos
```

```bash
  -d '{

    "prompt": "猫咪听歌摇头晃脑，下大雨",

      "output_config": {

```

```python
import requests

resp = requests.post(
    "https://www.geeknow.top/v1/videos",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json",
    },
    json={
        "model": "sora-2",
        "prompt": "猫咪听歌摇头晃脑，下大雨",
        "size": "720x1280",
        "seconds": "10",
        "input_reference": "data:image/png;base64,BASE64_IMAGE",
        "metadata": {
            "output_config": {
                "aspect_ratio": "9:16",
                "audio_generation": "Disabled",
                "resolution": "720P",
            }
        },
    },
    timeout=60,
)
print(resp.json())
```

## Body

<ParamField body="model" type="string" required>
模型名称。当前只整理 `sora-2`。
</ParamField>

<ParamField body="prompt" type="string" required>
提示词。
</ParamField>

<ParamField body="size" type="string">
尺寸字符串。当前常见值是 `720x1280` 和 `1280x720`；网关里 `sora-2` 当前也只显式校验这两种尺寸。
</ParamField>

<ParamField body="seconds" type="string | integer">
目标秒数。当前通过 `seconds` 提交；网关会兼容字符串和整数。
</ParamField>

<ParamField body="input_reference" type="string | array<string>">
参考图输入。当前 JSON 请求可使用 URL、base64、data URI；网关侧也兼容 multipart 文件上传。
</ParamField>

<ParamField body="metadata" type="object">
扩展配置。当前常见写法是 `metadata.output_config`，可放 `aspect_ratio`、`audio_generation`、`resolution` 等字段。
</ParamField>

<ParamField body="character_url" type="string">
Sora 角色创建所需视频链接。这个字段在通用 DTO 中存在，但从当前路由设计看，更适合配合 `/sora/v1/characters` 使用，而不是作为 `POST /v1/videos` 主链路字段。
</ParamField>

<ParamField body="character_timestamps" type="string">
角色片段时间范围，格式如 `1,3`。同样更适合角色创建链路。
</ParamField>

## 当前规则

| 项目 | 规则 |
| --- | --- |
| 默认尺寸 | 未显式传入时，按 `aspect_ratio` 推导横屏或竖屏尺寸 |
| `sora-2` 尺寸校验 | 当前网关只显式接受 `720x1280`、`1280x720` |
| 参考图格式 | JSON 下可用 URL、base64、data URI；multipart 也仍被网关兼容 |
