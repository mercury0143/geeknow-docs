# Sora 视频

Sora 系列当前使用 `POST /v1/videos` 创建视频任务，并通过 `GET /v1/videos/{task_id}` 查询任务状态和结果。

- 请求体推荐使用 JSON。
- 文生视频使用 `model`、`prompt`、`size`、`seconds`。
- 首帧/参考图生视频使用 `input_reference`。
- 比例和分辨率使用顶层 `size`。

## 当前模型

- `sora-2`
- `sora-2-pro`

## 当前接口行为

| 项目 | 当前行为 |
| --- | --- |
| 创建接口 | `POST /v1/videos` |
| 查询接口 | `GET /v1/videos/{task_id}` |
| 请求格式 | JSON |
| 参考图字段 | `input_reference` |
| 尺寸字段 | 顶层 `size`，例如 `1280x720` 或 `720x1280` |
| 时长字段 | 顶层 `seconds`，也兼容 `duration` |
