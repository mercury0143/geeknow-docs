# 文字系列

文字系列聚合 OpenAI 兼容格式、Claude 原生格式、Gemini 原生格式，以及模型列表、Embedding、Rerank、审核、网页读取与搜索等文本相关能力。

## 认证方式

大多数文字接口使用 Bearer Token：

```http
Authorization: Bearer YOUR_API_KEY
```

Claude 原生格式也兼容 Anthropic 风格请求头：

```http
x-api-key: YOUR_API_KEY
anthropic-version: 2023-06-01
```

Gemini 原生格式也兼容 Google API Key 风格：

```http
x-goog-api-key: YOUR_API_KEY
```

## 通用错误码

| 状态码 | 说明 |
| --- | --- |
| `400` | 请求参数错误，例如模型名称、消息结构或工具定义不合法 |
| `401` | API Key 缺失、无效或已过期 |
| `402` | 账户额度不足 |
| `429` | 请求频率超限 |
| `500` | 服务端内部错误 |
| `503` | 上游服务不可用或暂时过载 |

<Note>
文字 relay 路由会经过 TokenAuth、模型限流和渠道分发。模型实际可用参数取决于上游渠道能力，文档中标注为“可选”的字段不代表所有模型都会接受。
</Note>
