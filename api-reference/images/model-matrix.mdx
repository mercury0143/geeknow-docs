# 图像模型支持矩阵

当前对外图像能力并不只有单一的 `gpt-image-*` 模型。图像侧覆盖 Gemini、豆包 Seedream、Grok Image、GPT Image 与 OpenAI Images 兼容模型等多条模型线，它们对入口、参考图、尺寸和返回格式的要求并不完全一致。

<Info>
如果你只想找“该用哪个接口”，优先看“推荐入口”这一列；如果你只想找“这个模型能不能图生图”，优先看“参考图输入”这一列。
</Info>

## 模型总表

| 模型家族 | 代表模型 | 推荐入口 | 参考图输入 | 典型返回 |
| --- | --- | --- | --- | --- |
| Gemini 图像模型 | `gemini-3-pro-image-preview`、`gemini-2.5-flash-image-preview`、`gemini-3.1-flash-image-preview`、`gemini-3.1-flash-lite-image` | `/v1beta/models/{model}:generateContent` 或 `/v1/images/generations/async` | `inlineData` 多 part；异步入口使用 JSON `image` | `inlineData`、URL 或异步任务 ID |
| gpt-image-2 | `gpt-image-2` | `/v1/images/generations` | JSON `image`，支持多张 Base64 | `url` 或 `b64_json` |
| gpt-image-2-pro | `gpt-image-2-pro` | `/v1/images/generations` | JSON `image`，支持多张 Base64 | `url` 或 `b64_json` |
| gpt-image-2-vip | `gpt-image-2-vip` | `/v1/images/generations` | JSON `image`，支持多张 Base64 | `url` 或 `b64_json` |
| 图像异步生成 | `gpt-image-2`、`gpt-image-2-pro`、`gpt-image-2-vip`、Doubao Seedream、Gemini 图像模型 | `/v1/images/generations/async` + 查询接口 | JSON `image`，按目标模型填写字段 | 任务 ID，完成后返回 `url` 或 `b64_json` |
| 豆包 Seedream | `doubao-seedream-4-0-250828`、`doubao-seedream-4-5-251128`、`doubao-seedream-5-0-260128` | `/v1/images/generations` 或 `/v1/images/generations/async` | JSON `image`，通常为 Base64 数组 | `url`、`b64_json` 或异步任务 ID |
| Grok Image | `grok-4-2-image` | `/v1/images/generations` | JSON `image`，通常为 Base64 数组 | 主要是 `url` |
| OpenAI Images 兼容 | `gpt-image-1`、`dall-e-3`、`dall-e-2` | `/v1/images/generations`、`/v1/images/edits` | `image` / `mask` | `url` 或 `b64_json` |

## 分家族说明

### Gemini 图像模型

| 项目 | 说明 |
| --- | --- |
| 推荐接口 | `POST /v1beta/models/{model}:generateContent` |
| 模型示例 | `gemini-3-pro-image-preview`、`gemini-2.5-flash-image-preview`、`gemini-3.1-flash-image-preview`、`gemini-3.1-flash-lite-image` |
| 请求风格 | Gemini 原生 `contents[].parts[]` |
| 参考图 | 通过 `inlineData` 传图片二进制内容 |
| 比例参数 | `generationConfig.imageConfig.aspectRatio` |
| 尺寸参数 | `generationConfig.imageConfig.imageSize` |
| 尺寸特点 | `gemini-3-pro-image-preview` 支持 `1K` / `2K`；`gemini-2.5-flash-image-preview`、`gemini-3.1-flash-image-preview` 与 `gemini-3.1-flash-lite-image` 会回落为 `1K` |
| 异步接口 | `POST /v1/images/generations/async`，使用 OpenAI Images 兼容 JSON |
| 异步查询 | `GET /v1/images/generations/async/{taskId}` |
| 详细页面 | [Gemini 图像生成 API](/api-reference/images/gemini-image/generation) |

### gpt-image-2

| 项目 | 说明 |
| --- | --- |
| 推荐接口 | `POST /v1/images/generations` |
| 模型示例 | `gpt-image-2` |
| 请求风格 | OpenAI Images 兼容 JSON |
| 参考图 | `image: ["<base64>", ...]` |
| 返回格式 | `response_format = url` 或 `b64_json` |
| 异步接口 | `POST /v1/images/generations/async` |
| 异步查询 | `GET /v1/images/generations/async/{taskId}` |
| 尺寸特点 | 使用本模型尺寸表 |
| 详细页面 | [gpt-image-2 生成图像 API](/api-reference/images/gpt-image-2/generation) |

### gpt-image-2-pro

| 项目 | 说明 |
| --- | --- |
| 推荐接口 | `POST /v1/images/generations` |
| 模型示例 | `gpt-image-2-pro` |
| 请求风格 | OpenAI Images 兼容 JSON |
| 参考图 | `image: ["<base64>", ...]` |
| 返回格式 | `response_format = url` 或 `b64_json` |
| 异步接口 | `POST /v1/images/generations/async` |
| 异步查询 | `GET /v1/images/generations/async/{taskId}` |
| 尺寸特点 | 使用 `2K`、`4K` 高分辨率尺寸表 |
| 详细页面 | [gpt-image-2-pro 生成图像 API](/api-reference/images/gpt-image-2-pro/generation) |

### gpt-image-2-vip

| 项目 | 说明 |
| --- | --- |
| 推荐接口 | `POST /v1/images/generations` |
| 模型示例 | `gpt-image-2-vip` |
| 请求风格 | OpenAI Images 兼容 JSON |
| 参考图 | `image: ["<base64>", ...]` |
| 返回格式 | 由通道返回 `url` 或 `b64_json`；不建议依赖 `response_format` 强制切换 |
| 异步接口 | `POST /v1/images/generations/async` |
| 异步查询 | `GET /v1/images/generations/async/{taskId}` |
| 尺寸特点 | 使用独立 `1K`、`2K`、`4K` 表；每条边必须是 `16px` 倍数，最长边不超过 `3840px`，长短边比例不超过 `3:1`，总像素数在 `655,360` 到 `8,294,400` 之间 |
| 详细页面 | [gpt-image-2-vip 生成图像 API](/api-reference/images/gpt-image-2-vip/generation) |

### 图像异步生成

| 项目 | 说明 |
| --- | --- |
| 推荐接口 | `POST /v1/images/generations/async` |
| 模型示例 | `gpt-image-2`、`gpt-image-2-pro`、`gpt-image-2-vip`、`doubao-seedream-4-0-250828`、`doubao-seedream-4-5-251128`、`doubao-seedream-5-0-260128`、`gemini-3-pro-image-preview`、`gemini-2.5-flash-image-preview`、`gemini-3.1-flash-image-preview`、`gemini-3.1-flash-lite-image` |
| 请求风格 | OpenAI Images 兼容 JSON |
| 参考图 | `image: ["<url-or-base64>", ...]` |
| 字段范围 | 按目标模型填写 `model`、`prompt`、`n`、`size`、`image`、`quality` 等字段 |
| 查询接口 | `GET /v1/images/generations/async/{taskId}` |
| 结果获取 | 提交后保存任务 ID，查询完成后读取 `data[].url` 或 `data[].b64_json` |
| 适用场景 | 高分辨率、参考图或耗时较长的图像生成任务 |
| 详细页面 | [提交任务](/api-reference/images/gpt-image-async/generation) / [查询任务](/api-reference/images/gpt-image-async/query) |

### 豆包 Seedream

| 项目 | 说明 |
| --- | --- |
| 推荐接口 | `POST /v1/images/generations` |
| 模型示例 | `doubao-seedream-4-0-250828`、`doubao-seedream-4-5-251128`、`doubao-seedream-5-0-260128` |
| 请求风格 | OpenAI Images 兼容 JSON |
| 参考图 | `image: ["<base64>", ...]` |
| 比例到尺寸映射 | `16:9 -> 2560x1440`、`9:16 -> 1440x2560`、`1:1 -> 2048x2048` 等 |
| 返回格式 | 优先 `b64_json`，也可能只给 `url` |
| 异步接口 | `POST /v1/images/generations/async` |
| 异步查询 | `GET /v1/images/generations/async/{taskId}` |
| 详细页面 | [Doubao Seedream 生成图像 API](/api-reference/images/doubao-seedream/generation) |

### Grok Image

| 项目 | 说明 |
| --- | --- |
| 推荐接口 | `POST /v1/images/generations` |
| 模型示例 | `grok-4-2-image` |
| 请求风格 | OpenAI Images 兼容 JSON |
| 参考图 | `image: ["<base64>", ...]` |
| 比例到尺寸映射 | `16:9 -> 2560x1440`、`9:16 -> 1440x2560`、`1:1 -> 2048x2048` 等 |
| 返回格式 | 主要返回 `url`，不应依赖 `b64_json` |

### OpenAI Images 兼容

| 项目 | 说明 |
| --- | --- |
| 推荐接口 | `POST /v1/images/generations`、`POST /v1/images/edits` |
| 模型示例 | `gpt-image-1`、`dall-e-3`、`dall-e-2` |
| 请求风格 | OpenAI Images 原生字段 |
| 参考图 | `image`、`mask` |
| 返回格式 | `url` 或 `b64_json` |
| 注意事项 | 这组模型的公共语义见 [OpenAI Images 兼容概览](/api-reference/images/gpt-image-1/overview)，适合做兼容接入 |

## 比例与尺寸

### 常见比例

| 比例 | 常见家族 |
| --- | --- |
| `1:1` | Gemini、gpt-image-2、gpt-image-2-pro、gpt-image-2-vip、Seedream、Grok |
| `16:9` | Gemini、gpt-image-2、gpt-image-2-pro、gpt-image-2-vip、Seedream、Grok |
| `9:16` | Gemini、gpt-image-2、gpt-image-2-pro、gpt-image-2-vip、Seedream、Grok |
| `4:3` | gpt-image-2、gpt-image-2-pro、gpt-image-2-vip、Seedream、Grok |
| `3:4` | gpt-image-2-vip、Seedream、Grok |
| `3:2` / `2:3` | gpt-image-2、gpt-image-2-pro、gpt-image-2-vip、Seedream、Grok |
| `21:9` | gpt-image-2-vip、Seedream、Grok |
