# 图像系列

图像能力目前主要分成五类：按模型分组的 Gemini 官方生图入口、按模型分组的 GPT Image 生成入口、Doubao Seedream 统一生图入口、OpenAI Images 兼容生成/编辑入口，以及 Midjourney 专用任务接口。

  <Card title="模型支持矩阵" icon="table-cells" href="/api-reference/images/model-matrix">
    按模型家族查看图像接口支持范围、参考图输入方式、比例和尺寸约束。
  </Card>
  <Card title="Gemini 图像生成" icon="image" href="/api-reference/images/gemini-image/generation">
    `POST /v1beta/models/{model}:generateContent`，统一覆盖 `gemini-3-pro-image-preview`、`gemini-2.5-flash-image-preview`、`gemini-3.1-flash-image-preview`。
  </Card>
  <Card title="gpt-image-2" icon="image-plus" href="/api-reference/images/gpt-image-2/generation">
    `POST /v1/images/generations`，按基础比例和尺寸档位调用 `gpt-image-2`。
  </Card>
  <Card title="gpt-image-2-pro" icon="sparkles" href="/api-reference/images/gpt-image-2-pro/generation">

  </Card>
  <Card title="OpenAI Images 兼容" icon="wand-sparkles" href="/api-reference/images/gpt-image-1/overview">

  <Card title="Midjourney" icon="wand-sparkles" href="/api-reference/tasks/midjourney-task">

  <Card title="上传与结果查询" icon="upload" href="/api-reference/uploads/image-upload">

## 路由清单

| Method | Path | 说明 |
| --- | --- | --- |
| `POST` | `/v1/images/generations` | 统一图像生成入口 |
| `POST` | `/v1/images/edits` | OpenAI Images 兼容编辑入口 |
| `POST` | `/v1/edits` | 旧版编辑别名 |
| `POST` | `/v1/images/variations` | 路由已保留，当前未实现 |
| `POST` | `/mj/submit/imagine` | Midjourney imagine |
| `POST` | `/mj/submit/blend` | Midjourney blend |
| `POST` | `/mj/submit/describe` | Midjourney describe |
| `POST` | `/mj/submit/edits` | Midjourney 编辑相关动作 |
| `POST` | `/mj/submit/video` | Midjourney 视频相关动作 |
| `GET` | `/mj/task/{id}/fetch` | Midjourney 任务查询 |
| `GET` | `/mj/image/{id}` | Midjourney 图片代理 |
| `POST` | `/v1beta/models/{model}:generateContent` | Gemini 官方格式生图入口 |

<Warning>
`/v1/images/variations`、`/v1/files*` 和 `fine-tunes*` 这类 OpenAI 兼容路径虽然在路由中占位，但当前会返回未实现错误，不能直接用于生产。
</Warning>
