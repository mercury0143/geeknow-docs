# 音频系列

音频相关能力分成两部分：一部分是 `/v1/audio/*` 下的 OpenAI Audio 兼容接口，另一部分是 `/suno/*` 下的异步音乐任务接口。

  </Card>
  <Card title="Suno 音乐生成" icon="music" href="/api-reference/audio/suno">
    `/suno/submit/{action}`、`/suno/fetch`、`/suno/fetch/{id}`。
  </Card>

## 路由清单

| Method | Path | 说明 |
| --- | --- | --- |
| `POST` | `/v1/audio/transcriptions` | 语音转文字 |
| `POST` | `/v1/audio/translations` | 语音翻译为英文 |
| `POST` | `/v1/audio/speech` | 文字转语音 |
| `POST` | `/suno/submit/{action}` | Suno 提交任务 |
| `POST` | `/suno/fetch` | Suno 批量查询 |
| `GET` | `/suno/fetch/{id}` | Suno 单任务查询 |
